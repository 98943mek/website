Heapsort

Brief Details

The heapsort algorithm takes advantage of the heap data structure. In the first step, the array is heapified. Then, it pops the top of this heap and add the top element to the array until the array is full and the heap is empty. Each operation to the heap takes O(logn) time. So this algorithm yields O(nlogn) time complexity in worst case.

Time Complexity :

Best Case: O(n)
Average Case: O(nlogn)
Worst Case: O(nlogn)

void heapsort(int n)
{
    int i,c,tmp;

    for(i = 1; i < n; i++)
    {
        c = i;

        while(c != 0)
        {
            if(inp[c] > inp[(c >> 1)])
            {
                tmp = inp[c];
                inp[c] = inp[(c >> 1)];
                inp[(c >> 1)] = tmp;

                c >>= 1;
            } else {
                break;
            }
        }
    }

    for(i = n - 1; i > 0; i--)
    {
        tmp = inp[i];
        inp[i] = inp[0];
        inp[0] = tmp;

        c = 0;

        while(c < i)
        {
            if((c << 1) + 2 < i)
            {
                if(inp[c] < max(inp[(c << 1) + 1],inp[(c << 1) + 2]))
                {
                    if(inp[(c << 1) + 1] > inp[(c << 1) + 2])
                    {
                        tmp = inp[c];
                        inp[c] = inp[(c << 1) + 1];
                        inp[(c << 1) + 1] = tmp;

                        c = (c << 1) + 1;
                    } else {
                        tmp = inp[c];
                        inp[c] = inp[(c << 1) + 2];
                        inp[(c << 1) + 2] = tmp;

                        c = (c << 1) + 2;
                    }
                } else {
                    break;
                }
            } else if((c << 1) + 1 < i) {
                if(inp[c] < inp[(c << 1) + 1])
                {
                    tmp = inp[c];
                    inp[c] = inp[(c << 1) + 1];
                    inp[(c << 1) + 1] = tmp;

                    c = (c << 1) + 1;
                }

                break;
            } else {
                break;
            }
        }
    }
}
