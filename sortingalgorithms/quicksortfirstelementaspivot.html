Quicksort - First Element as Pivot

Brief Details

The quicksort algorithm is a divide-and-conquer algorithm. In each step it chooses a pivot, as for the this implementation it chooses the first element. Then it divides the array into 2 sets, one smaller than the pivot and one larger. Then the algorithm recurses into those 2 subarrays.

Time Complexity :

Best Case: O(nlogn)
Average Case: O(nlogn)
Worst Case: O(n2)

void quicksort_first_element_as_pivot(int s, int e)
{
    if(s >= e)
    {
        return;
    }

    int piv = inp[s],fp = s,lp = e,tmp;

    while(fp < lp)
    {
        while(fp < lp && inp[fp] < piv)
        {
            fp++;
        }

        while(fp < lp && inp[lp] > piv)
        {
            lp--;
        }

        if(fp < lp)
        {
            tmp = inp[fp];
            inp[fp] = inp[lp];
            inp[lp] = tmp;
            fp++;
            lp--;
        }
    }

    if(fp == lp)
    {
        quicksort_first_element_as_pivot(s,fp);
        quicksort_first_element_as_pivot(fp + 1,e);
    } else {
        quicksort_first_element_as_pivot(s,lp);
        quicksort_first_element_as_pivot(fp,e);
    }
}
